ctrl + l: Limpa o terminal

ls : Lista o que contem dentro do diretorio 
cd :  Muda de diretorio
pwd : Mostra o caminho do diretorio que esta
mkdir : Cria uma pasta nova
ls -a : Mostra arquivos ocultos
cd .. : Volta uma pasta acima
cat : 	Le o que contem no arquivo
git init . : Cria uma pasta git, um repositorio.
rm -rf .git: Exclue a pasta git
git status: Mostra como esta o diretorio
git add . : Adiciona o que queremos que ira para o commit (ele prepara/sinaliza o que ira entrar no "chekpoint")
git commit -m: 	Mensagem que explica o que você fez
git config --global user.email: Configurar o e-mail que vai aparecer nos commits
git config --global user.name: Configurar o nome que vai aparecer nos commits
git log: Mostra todos o historico de commit que foi salvo
. : Pasta Atual. dietório atual
* : Tudo do diretorio atual
git diff : Diferença do estado atual em relação ao commit anterior
git reset: Volta no tempo (desfaz commit/alteração)
git checkout: Você muda para outra branch.
git checkout -b : Aqui você cria uma branch nova e já entra nela.
git branch: Só lista as branchs que existem no seu repositorio local.
git merge: Mescla as blanch, as blanch informada
teste
